URL-Expander / URL-Unshortener

urlex.org is a URL Expander that unshortens / deshortens any URL to a long URL masked by any URL-Shortener.

By clicking unshortened links, the HTTP-Referer header ist set to http://urlex.org for your safety.

Note that you can unshorten multiple URLs at once by entering just one per line.

Billions of URLs Expanded

This service already expanded a ton of short-links and unveiled millions of paywall links.

It is the fastest and easiest to use service for safely demask shortened or obfuscated URLs.

Proudly built with the power of Node.js

Batch Requests (Pro!)

You can only access our API with a Pro account.

Unshorten URL by simply appending to main address, like http://urlex.org/http://bit.ly/iiLcUI (works without Pro account)

Batch expansion (separated by ***): http://urlex.org/http://bit.ly/iiLcUI***http://bit.ly/iiLcUI

Alternative output formats (txt, json, xml): http://urlex.org/xml/http://bit.ly/iiLcUI

Define custom timeouts (in seconds) to speed up expansion or extend slowly loading requests.

Get all hops (redirects) within JSON and XML output and follow the entire path a URL makes from beginning till the end.

This service is free. However, it is limited to 1,000 requests per day.

Do not exploit it excessively, otherwise your requests might get blocked permanently.
